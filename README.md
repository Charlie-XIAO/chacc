# chacc

`chacc` is yet another small C compiler, but written in Rust, mostly for fun and learning purposes. It is largely inspired by and references [`chibicc`](https://github.com/rui314/chibicc) (which is a small yet powerful C compiler written in C).

## Setup

- Install [Rust](https://rust-lang.org/tools/install/) toolchain (recommended 1.90+)
- (Optional) Install [just](https://just.systems/man/en/) task runner
- (Optional) Install [cargo-nextest](https://nexte.st/docs/installation/pre-built-binaries/) test runner
